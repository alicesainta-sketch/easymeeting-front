# EasyMeeting

EasyMeeting 是一个基于 Electron + Vue 3 的桌面端纯前端项目，当前已具备登录演示与会议管理核心流程。

## 项目功能清单

### 1) 桌面壳能力

- 自定义窗口标题栏（最小化、最大化/还原、关闭）。
- 登录/会议工作区切换时窗口模式联动。

### 2) 登录与鉴权

- 登录/注册模式切换（含窗口高度切换）。
- 登录表单校验：邮箱格式、昵称长度、密码长度、二次密码一致、验证码校验。
- 本地验证码生成与刷新（前端 SVG，无后端依赖）。
- 登录/注册提交流程演示（loading + 操作提示）。
- 路由守卫：未登录无法访问会议页面，登录后自动进入会议中心。

### 3) 会议列表与管理

- 会议列表展示（标题、主题、房间号、主持人、时间范围、倒计时）。
- 搜索过滤（标题/主题/房间号）。
- 状态筛选（全部/进行中/待开始/已结束）。
- 排序切换（智能排序 / 开始时间升序 / 开始时间降序）。
- 新建会议（本地写入）。
- 编辑会议、删除会议（二次确认）。
- 复制会议：
  - 列表复制后自动打开副本编辑弹窗。
  - 详情页支持复制并跳转副本详情。
- 会议表单校验：开始时间必须晚于当前时间，时长限制 15-180 分钟。

### 4) 会议详情与提醒

- 详情信息展示（基础信息、议程、参会人、备注）。
- 详情操作（进入会议、编辑、复制会议、删除、复制房间号）。
- 倒计时实时刷新（列表/详情同步）。
- 本地提醒能力：
  - 手动“提醒我”提示。
  - 自动提醒（距离开始 10 分钟 / 1 分钟）。

### 5) 会议房间（纯前端演示）

- 进入会议路由：`/meetings/:id/room`。
- 会前检查页面：
  - 本地摄像头/麦克风预览（`getUserMedia`）。
  - 摄像头/麦克风开关。
  - 输入设备选择与昵称设置。
- 房间页面：
  - 本地画面展示与参会人占位网格。
  - 控制栏（开关麦克风、开关摄像头、离开会议）。

### 6) 本地持久化

- 登录状态与会议数据通过 `electron-store` 持久化保存。
- 非 Electron 场景自动回退到 `localStorage`，保证纯前端可运行。

## 说明

- 本项目定位为纯前端项目，不对接后端服务。
- 登录/注册提交结果使用本地演示逻辑，不依赖远程接口。
- 会议房间的摄像头/麦克风本地预览不需要后端。
- 若要实现 Zoom 类多人实时音视频，需要后端信令服务与 STUN/TURN 支持。

## 技术栈

- Electron
- Vue 3
- Element Plus
- Electron-Vite
- Vue Router
- ESLint + Prettier

## 快速开始

### 安装依赖

```bash
npm install
```

### 启动开发环境

```bash
npm run dev
```

### 代码检查

```bash
npx eslint . --no-cache --config eslint.config.mjs
```

### 构建

```bash
npm run build
```

### 平台打包

```bash
npm run build:win
npm run build:mac
npm run build:linux
```
