# Metrics

## 性能指标
- **首屏可交互时间（TTI）**：<= 3s
- **会议列表加载耗时**：<= 800ms
- **房间进入耗时**：<= 1.5s

## 稳定性指标
- **事件写入失败率**：< 0.1%
- **回放构建失败率**：< 0.1%
- **异常恢复成功率**：>= 99%

## 体验指标
- **操作可见反馈延迟**：<= 100ms
- **提示文案可读性**：所有关键操作需有明确状态反馈

## 采集口径
- 客户端记录关键流程耗时与错误码。
- 会议信息与事件流统计聚合至本地指标。

## 告警阈值
- 任一核心流程失败率连续 3 次超过阈值时触发告警。
